# Technical SEO

## Beschreibung
Crawlbarkeit, Indexierung, Redirects, Sitemaps, Ladezeit/Core Web Vitals, strukturierte Daten (technisch), Health Score. Wöchentlicher Crawl über `seo@eom.de`.

## Standard-Checks
- 404/Broken Links, Redirect-Ketten, 302->301, 5XX in Sitemap
- Canonicals, hreflang, noindex-Logik, Parameter-/Rechner-URLs
- Missing Alt-Text, Orphan Pages, Slow Pages, Twitter-/X-Cards
- Redirect-Regel: defekte Ziele 301 auf passende Live-Seite, **nie** auf Startseite (Soft-404)

- VHV „Druckstücke"-404s + Legacy-Redirects
