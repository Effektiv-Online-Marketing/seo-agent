# Über EOM

## Wer ist EOM
EOM (Effektiv Online Marketing) ist eine Performance-Marketing-Agentur, die SEO- (und teils SEA-/Social-) Mandate für Mittelstand und Konzerne im DACH-Raum betreut. Kernprinzip: **eine** standardisierte Output-Architektur (Reporting, Analyse, Strategie-Scan), die pro Kunde nur befüllt wird — „ein Template, viele Kunden".

## Mandatstypen
- **Kollaborativ / direkter Doc-Zugriff** (Beispiel NIBE): Content-Review direkt im Google Doc mit dem Kunden (Grün = neuer Vorschlag, Gelb = Bestand anpassen).
- **Empfehlungs-/Übergabemodell** (Beispiel VHV): kein CMS-Schreibzugriff, Output = übergabefertige Empfehlung (PDF + DOCX), Umsetzung kundenseitig (bei VHV via Dienstleister Neoskop in Magnolia auf Stage).
- **Retainer mit Stundenbudget** (Beispiel VHV): Toggl-Zeiterfassung gegen Monatsbudget, Priorisierung zählt.

## Standard-Cadence (typisch)
- Wöchentlich: technisches Crawl-Audit (`seo@eom.de`)
- Monatlich: Keyword- + Backlink-Report, Monatsreporting-Dokument, Looker-Dashboard
- ~14-tägig: Jour Fixe mit Kunde
- Jährlich/anlassbezogen: Strategie-Scan & Quick-Check
